# Privacy Policy

Effective date: 2026-05-19

Markdown Composer for ChatGPT is a local browser extension that adds a Markdown editing overlay to the ChatGPT website.

## Data collection

This extension does not collect, transmit, sell, or share user data.

## Local storage

The extension may store draft text and UI preferences in the browser's local storage on the ChatGPT website. This data remains on the user's device unless the user separately sends the text through ChatGPT's native input box.

## Network requests

The extension does not make external network requests and does not use a backend server.

## Third-party affiliation

This extension is not affiliated with, endorsed by, or sponsored by OpenAI.
